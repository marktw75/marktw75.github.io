安裝 Link to heading

大概做一下筆記

安裝Hugo程式，下載Coder主題。將主題解壓縮後，再使用Eclipse編輯。 編輯的過程，Hugo server可以即時呈現改好的內容，十分方便。

C:\Users\markt\dev\exampleSite>hugo server

C:\Users\markt\dev\exampleSite>hugo
